# Teen Mental Health Dashboard

## Project Overview

The Teen Mental Health Dashboard is a data analysis project designed to explore the relationship between social media usage, sleep patterns, stress levels, anxiety, and addiction among teenagers. The dashboard delivers visual insights to help understand how digital behavior impacts mental health.

## Objectives

- Analyze teen mental health trends.
- Examine the impact of social media usage on stress and anxiety.
- Study the relationship between sleep duration and mental well-being.
- Compare mental health indicators across gender groups.
- Identify patterns that support awareness and decision-making.

## Dataset Features

The dataset includes the following features:

- Age
- Gender
- Daily Social Media Hours
- Sleep Hours
- Screen Time Before Sleep
- Stress Level
- Anxiety Level
- Addiction Level
- Platform Usage (Instagram, TikTok, Both)

## Dashboard Components

### 1. Gender Analysis

This section compares average metrics for male and female teenagers, including:

- Sleep Hours
- Social Media Usage
- Screen Time Before Sleep

### 2. Sleep Category Analysis

This section contrasts mental health indicators across sleep categories:

- Good Sleep
- Normal Sleep
- Poor Sleep

It highlights differences in:

- Stress Level
- Anxiety Level
- Addiction Level
- Sleep Hours

### 3. Social Media Platform Analysis

This component investigates relationships between platform usage and mental health, focusing on:

- Platform Usage
- Daily Social Media Hours
- Stress Level

### 4. KPI Cards

Key performance indicators shown in the dashboard include:

- Average Stress Level
- Average Sleep Hours
- Sleep vs Stress Correlation

### 5. Interactive Filters

The dashboard supports dynamic filtering by:

- Gender
- Age
- Social Media Usage Category

## Key Insights

- Sleep and stress show a negative correlation, suggesting lower sleep hours are associated with higher stress levels.
- Teenagers with poor sleep tend to experience higher stress, anxiety, and addiction levels.
- Social media usage patterns vary by platform and demographic group.
- Mental health indicators differ across age and gender categories.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Dashboard Design
- Data Cleaning and Transformation

## Conclusion

The Teen Mental Health Dashboard provides a comprehensive view of teen mental health and highlights the influence of sleep habits and social media usage on psychological well-being. The findings can support educators, parents, and researchers in understanding challenges faced by teenagers.
