# Teen Mental Health Dashboard

Simple Streamlit dashboard for `Teen_Mental_Health_Dataset.csv`.

## Setup

Create a virtualenv and install dependencies:

```bash
python -m venv .venv
source .venv/bin/activate   # or .venv\Scripts\activate on Windows
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```

## Notes
- The app auto-detects columns and handles duplicates/missing values.
- Ask me to add specific charts or filters you want.
